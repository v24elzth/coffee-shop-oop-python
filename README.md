# ☕ Coffee Shop CLI – OOP + SQLite

Command-line coffee shop application built using Object-Oriented Programming, SQLite database, Observer Pattern, and Dependency Injection.

This project simulates a real coffee shop system with stock management, promotions, and customer tracking.

---

## 🚀 Features

- Order coffee from menu
- SQLite database storage
- Stock management
- Promotion system (Observer Pattern)
- Top 5 customers report
- Sales history
- DTO architecture
- Dependency Injection Container

---

## 🧠 Concepts Used

- OOP (Class, Composition, Interface)
- Abstract Base Class (ABC)
- Observer Pattern
- Dependency Injection
- SQLite Persistence Layer
- DTO (Data Transfer Object)
- CLI UI Layer
